How to setup Pelican

Mon 04 November 2013

最近決定要來寫一下blog, 看到有人推薦pelican, 就決定來研究一下
剛好pelican是用python寫的, 就決定是你啦～

先安裝python 套件pip和virtualenv virtualenvwrapper
virtualenv在機器上模擬一個環境以便開發 

sudo apt-get install python-pip
sudo pip install virtualenv virtualenvwrapper


vi ~/.bashrc
# Virtualenvwrapper
export WORKON_HOME=$HOME/.virtualenvs
source /usr/local/bin/virtualenvwrapper.sh

搞定了之後就利用virtualenvwrapper模擬一個環境 

mkvirtualenv blog
workon blog

移動到資料夾, 並安裝pelican, markdown 

cd ~/.virtualenvs/blog
pip install pelican markdown
pip install --upgrade pelican markdown

準備工作到這邊已經差不多都做完了
已經安裝了可以寫blog的工具
再來可以先上github開一個repository, github可以幫忙我們把靜態網頁掛在上面
如果repository的名字是 username.github.io 表示這是使用者的個人頁面
一個人只能申請一個, 靜態網頁要放在master branch
如果是一般名稱的話是project頁面, 每個project可以有一個
靜態網頁要放在gh-pages branch

開始blog設定

pelican-quickstart

這邊會問一連串的問題, 填完之後就完成整個系統的基本設定
之後的一些資料就直接放連結, 有點懶的寫了 XDDD
